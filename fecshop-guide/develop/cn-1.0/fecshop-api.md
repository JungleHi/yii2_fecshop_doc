Fecshop Api 介绍
===========

> fecshop api这个入口的定位是和第三方系统对接的api部分，譬如和ERP，
> 进行订单传递和产品刊登等，以及erp把订单的物流状态传递个fecshop网站
> 等等


### 配置：

fecshop appapi的配置文件为：`@fecshop/app/appapi/config/appapi.php`

本地appapi的配置文件路径为：`@appapi/config/`

由于这个部分的定制性很强，因此，这个部分只做产品和订单的api，
其他的可以自行根据需要自己定制开发。
